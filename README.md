# Algorithm Practice

- 알고리즘 코딩테스트 준비를 위한 개인 학습 기록
- CodeTree 플랫폼 문제들을 체계적으로 학습하고 해결한 코드 정리
- 각 날짜별로 학습한 문제들을 체계적으로 정리

## 학습 소개
- 현황
  - 다양한 난이도의 알고리즘 문제 해결
  - 체계적인 학습 기록 관리
- 향후 계획
  - 고급 알고리즘 학습 (그래프, 트리, 동적 프로그래밍)
  - 자료구조 심화 학습
  - 알고리즘 최적화 기법 습득
  - 실제 코딩테스트 응시 및 성과 향상

## 파일 구조
```
YYMMDD/
├── 문제명/
│   ├── README.md          # 문제 설명 및 링크
│   └── solution.py        # 해결 코드
└── README.md              # 해당 날짜 학습 현황
```

- Python 파일: 영어로 된 문제명을 하이픈으로 연결
- README 파일: 각 문제별 상세 설명과 CodeTree 링크 포함

## 사용 기술
- 프로그래밍 언어: Python 3
- 버전 관리: Git
